# 📊 Paris Olympics 2024 Analytics Dashboard

## 📌 Project Overview
The **Paris Olympics 2024 Analytics Dashboard** is a data analytics and visualization project that provides insights into Olympic athlete participation, medal distribution, and country performance.

The dashboard allows users to explore Olympic data through interactive charts, filters, and visual comparisons. It helps analyze athlete demographics, medal distribution, country-wise performance, and historical Olympic trends.

---

## 🚀 Features

### 🏅 Medal Analysis
- Gold, Silver, and Bronze medal distribution  
- Country-wise medal comparison  
- Medal trends across different nations  

### 🏃 Athlete Insights
- Athlete count by gender  
- Athlete distribution by age category  
- Country-wise athlete participation  

### 🌍 Country Performance
- Country-wise medal analysis  
- Interactive world map visualization  
- Medal comparison between countries  

### 📈 Historical Analysis
- Medal trends across Olympic years  
- Historical medal distribution analysis  

### 🔎 Interactive Filters
- Country search functionality  
- Sport category navigation  
- Year-based filtering  

---

## 🖥️ Dashboard Pages

### 🏠 Home Page
Provides an introduction and overview of the Paris Olympics analytics dashboard.

### 📊 Overview Dashboard
Displays key highlights including total athletes, teams, participating countries, and medal distribution.

### 🏃 Athletes Dashboard
Shows athlete demographics such as gender distribution and participation across different age categories.

### 🌍 Country Dashboard
Provides country-level performance analysis, including medal counts and an interactive world map visualization.

### 📈 Historical Analysis
Visualizes medal distribution across Olympic years to identify historical trends.

---

## 📊 Key Insights

- Total **11,113 athletes** participated in the Olympics  
- **1,698 teams** represented **206 countries**  
- Medal distribution across **Gold, Silver, and Bronze** categories  
- Country-wise performance comparison highlights top-performing nations
